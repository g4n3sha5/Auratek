# How to run?
1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run server 
   ```sh
   npm run dev
   ```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
